Install and Open Proof

Proof runs on macOS and is distributed as a Mac app. Open the DMG, drag Proof into Applications, then Control-click the app and choose Open the first time macOS warns you.

If macOS says the app is damaged, remove the quarantine attribute and open it again:

xattr -dr com.apple.quarantine /Applications/Proof.app

Create or Open a Catalog

A catalog is Proof's local database for your library. It stores folder references, thumbnails, previews, ratings, flags, color labels, keywords, collections, and edit metadata.

Your original files stay where they are. Proof references photos in place. Catalog work is separate from the originals unless you explicitly move, rename, export, or move files to Trash.

On first launch, create a new .proof catalog or open an existing one. Later, use the catalog menu in the top-left toolbar to switch catalogs, create another catalog, or reopen a recent catalog.

Add and Sync Folders

Click Add Folder and choose a folder of photos. Proof scans the folder and subfolders, reads metadata, creates thumbnails, and shows the folder in the left sidebar.

  • All Photographs shows every indexed photo in the catalog.
  • Added root folders can include expandable subfolders.
  • Offline and missing states appear when source folders or files are unavailable.

Use Rescan after files change outside Proof. A rescan adds new photos, refreshes changed metadata, marks missing files, and preserves existing ratings, flags, keywords, collections, and edit metadata.

Right-click a folder to reveal it in Finder, rescan it, reconnect an offline folder, rebuild thumbnails, remove missing catalog rows, or remove the folder from the catalog. Removing a folder from the catalog does not delete files from disk.

Culling works from Grid, Loupe, and Survey. Navigate through the shoot, then mark the current photo or current selection.

Enable Auto-advance in the filter bar or Preferences to jump to the next photo after flagging, rating, or labeling. Auto-advance is disabled for batch selection operations so the selected set remains stable.

Compare and Inspect

Loupe

Open a single photo with E, double-click, or Enter. Use fit through 400% zoom, the magnifier, the hand tool, and the metadata bar.

Survey

Select two or more photos, press N, and compare frames side by side with synced zoom and pan for focus checks and keeper selection.

Keywords are edited in the Loupe metadata bar. Add a keyword, press Enter, then use the filter bar to narrow the grid by keyword later.

Export, Delete, and Stay Safe

Export writes new rendered files and never modifies originals. Select one or more photos, choose Export, pick JPEG, PNG, TIFF, or WebP, and choose a destination file or folder.

Press Delete or Backspace to move selected photo files to the macOS Trash. Use Delete N Rejects after marking rejects with X to clear a batch. Trash moves are recoverable until the Trash is emptied.

Most Proof actions are non-destructive. The commands that intentionally affect files on disk are moving photos, renaming photos, moving photos to Trash, and Fuji .HIF to .HEIC extension normalization during indexing or rescan.
